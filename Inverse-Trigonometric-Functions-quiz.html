  • Question 2:  

    The value of cot(π/4 - 2cot-1 3) is

  • Correct Option is :

    3

  • Solution :

    Given, cot(π/4 - 2cot-1 3)

    = cot[π/4 - cot-1 {(32 - 1)/(2*3)}]

    = cot[π/4 - cot-1 {(9 - 1)/6}]

    = cot[π/4 - cot-1 {8/6}]

    = cot[π/4 - cot-1 {4/3}]

    = {cot(π/4) * (4/3) + 1}/{4/3 - cot(π/4)}

    = {1 * (4/3) + 1}/{4/3 - 1}

    = {4/3 + 1}/{4/3 - 1}

    = {7/3}/{1/3}

    = 7

  • Question 4:  

    If tan-1 (x - 1) + tan-1 x + tan-1 (x + 1) = tan-1 (3x), then the value of x are
  • Correct Option is :

    3

  • Solution :

    Given, tan-1 (x - 1) + tan-1 x + tan-1 (x + 1) = tan-1 (3x)

    => tan-1 (x - 1) + tan-1 (x + 1) + tan-1 x = tan-1 (3x)

    => tan-1 [{(x - 1) + (x + 1)}/{1 - (x - 1) * (x + 1)}] + tan-1 x = tan-1 (3x)

    => tan-1 [{2x}/{(1 - (x2 - 1)}] + tan-1 x = tan-1 (3x)

    => tan-1 [{2x}/{(1 - (x2 - 1)}] + tan-1 x = tan-1 (3x)

    => tan-1 [{2x}/{(1 - x2 + 1)}] + tan-1 x = tan-1 (3x)

    => tan-1 [{2x}/{(2 - x2 )}] + tan-1 x = tan-1 (3x)

    => tan-1 [{2x/(2 - x2 ) + x}/{1 - 2x2 /{(2 - x2 )}]  = tan-1 (3x)

    => tan-1 [{2x + x(2 - x2 )}/(2 - x2 )]/[{1 - 2x2 /{(2 - x2 )}] = tan-1 (3x)

    => tan-1 [{2x + x(2 - x2 )}/{(2 - x2 ) - 2x2 }] = tan-1 (3x)

    => tan-1 [{2x + 2x - x3 )}/{2 - x2  - 2x2 }]  = tan-1 (3x)

    => tan-1 [{4x - x3 }/{2 - x2  - 2x2 }] = tan-1 (3x)

    => tan-1 [{4x - x3 }/{2 - 3x2 }] = tan-1 (3x)

    => {4x - x3 }/{2 - 3x2 } = 3x

    => 4x - x3 = {2 - 3x2 ) * 3x

    => 4x - x3 = 6x - 9x3

    => 4x - x3 - 6x + 9x3 = 0

    => 8x3 - 2x = 0

    => 2x(4x2 - 1) = 0

    => 2x*(2x - 1)*(2x + 1) = 0

    => x = 0, 1/2, -1/2

    So, the value of x are 0, 1/2, -1/2

  • Question 5:  

    If cos-1 x + cos-1 y + cos-1 z = π, then x2 + y2 + z2 + 2xyz =

  • Correct Option is :

    2

  • Solution :

    Let cos-1 x = A, cos-1 y = B, cos-1 z = C

    Given, cos-1 x + cos-1 y + cos-1 z = π

     => A + B + C = π

    => A + B = π - C

    => cos(A + B) = cos(π - C)

    => cos A*cos B - sin A*sin B = -cos C

    => cos A*cos B + cos C = sin A*sin B

    => (cos A*cos B + cos C)2 = (sin A*sin B)2

    => sin2 A*sin2 B = (cos A*cos B + cos C)2

    => (1 - cos2 A)*(1 - cos2 B) = (cos A*cos B + cos C)2

    => (1 - x2 )*(1 - y2 ) = (xy + z)2

    => 1 - x2 - y2 + x2 y2 = x2 y2 + z2 + 2xyz

    => 1 = x2 + y2 + z2 + 2xyz

  • Question 9:  

    If tan(x + y) = 33 and x = tan-1 3, then tha value of y is

  • Correct Option is :

    1

  • Solution :

    Given, tan(x + y) = 33

    => x + y = tan-1 (33)

    => tan-1 3 + y = tan-1 33

    => y = tan-1 33 - tan-1 3

    => y = tan-1 {(33 - 3)/(1 + 33*3)}

    => y = tan-1 {30/(1 + 99)}

    => y = tan-1 {30/100}

    => y = tan-1 (0.3)
